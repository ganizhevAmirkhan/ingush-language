# Ингушско–Русский Словарь (Inguish–Russian Dictionary)

Этот проект содержит большой интегрированный словарь с переводами
и толкованиями ингушского языка.

## Структура проекта

- **ingush_dictionary_full.txt** — основной словарь:
  - Русское слово → ингушский перевод
  - Толкование на ингушском (если найдено)
  - Формат совместим с машинной обработкой

- Исходные данные:
  - `rus_to_ing.json` — большой набор русско-ингушских соответствий
  - `ingrus.txt` — толковый ингушский словарь (в латинице, конвертировано в кириллицу)

## Цель проекта

Создать самый полный, открытый и точный
ингушско-русский словарь по состоянию на сегодняшний день.

## Автоматическая сборка

Словарь был обработан программно:
- нормализация ключей
- сопоставление переводов
- сопоставление толкований
- фильтрация нерусских ключей
- строгие совпадения для толкований
- объединение всех значений в одну статью
